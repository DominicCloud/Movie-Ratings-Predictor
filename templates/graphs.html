Performance Metrics Visualized

Bar Chart of Performance Metrics

This bar chart displays the values for three key performance metrics: Mean Squared Error (MSE), Root Mean Squared Error (RMSE), and Mean Absolute Error (MAE). In this chart:

  • The MSE value is 0.0661, represented in a violet bar, indicating the average of the squared errors.
  • The RMSE value is 0.2572, shown as a yellow bar, which provides the error in the same units as the predictions.
  • The MAE value is 0.2000, in pink, representing the average absolute error between predictions and actual values.
A lower value for each metric indicates better model performance. RMSE and MAE values are close to each other, suggesting consistent error patterns.

Residuals Histogram

This histogram shows the distribution of residuals, or the differences between the actual values and the predicted values. Ideally, residuals should be centered around zero, with minimal spread. In this case:

  • The distribution is fairly symmetric around zero, which indicates that the model has low bias.
  • However, the slight spread of errors (visible in the histogram with a higher frequency of residuals between -0.25 and 0.25) suggests that there is still some room for improvement in minimizing prediction errors.
The histogram uses a dark violet color for the bars, giving a clear visualization of how the prediction errors are distributed.

Scatterplot of True vs Predicted Values

This scatterplot compares the actual values with the predicted values. Ideally, all points would lie on the diagonal line, which would indicate perfect predictions. In this plot:

  • Many of the points are close to the diagonal, suggesting that the model is performing reasonably well.
  • The scatter points are color-coded in shades of orange, and there is some spread away from the diagonal, indicating slight prediction errors in some cases.
  • This visualization provides a quick way to assess the overall accuracy of the model, with smaller spreads indicating better performance.
